# Customer Churn Prediction MLOps Project

## Overview
This project predicts customer churn using Machine Learning and exposes predictions through a FastAPI REST API.

## Tech Stack
- Python
- Scikit-learn
- FastAPI
- Docker
- Kubernetes
- GitHub

## Features
- Customer churn prediction model
- REST API with FastAPI
- Dockerized application
- Kubernetes deployment
- Real-time prediction testing
